# 09_rejected_or_uncertain

REVIEW.md lists every raw model that is rejected or uncertain, with the reason and
what would change the decision. This folder holds NO file copies (rejected bodies are
~40 MB); the originals stay in ../unsorted_stl_raw/ and are never deleted.

If a part is later un-rejected: move its row to MODEL_INVENTORY.md with the new
decision, copy the file into 02_ready_to_slice/ and add its MANIFEST.md row.
