# 06_finishing

Per-part finishing records (sanding, primer, paint, decals, clear coat), using the
"Finishing note" template in ../PRINT_LOG_TEMPLATE.md. Method lives in
../FINISHING_GUIDE.md; this folder records what was actually done to which part.

Naming: FIN-<part-name>.md  (e.g. FIN-body-front.md)
