# 03_print_profiles

Bambu Studio profile notes and exports. Starting values live in ../PRINT_SPEC.md;
this folder records what was ACTUALLY used once a profile is saved in Bambu Studio.

Naming: profile-<part-class>-<material>-vNN.md  (e.g. profile-body-panels-PLA-v01.md)
Each note: base preset used, every changed setting, which parts it applies to, date.
Optionally export the Bambu Studio preset (.json) alongside with the same name.
