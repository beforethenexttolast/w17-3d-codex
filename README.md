# w17-3d-codex — 3D printing & fabrication for the W17 RC F1 car

Everything needed to turn the raw STL bundles into a printed, painted, assembled
1/10-scale Mercedes W17 (#63) on the OpenRC RC-01 chassis. Printer: Bambu X1 Carbon.

**Start here:**
- New to the project → `GENERAL_PLAN.md` (roadmap + open questions)
- Never printed before → `BEGINNER_3D_PRINTING_GUIDE.md`
- What to print, in what order → `BUILD_SHEET.md`
- What each raw file is → `MODEL_INVENTORY.md`
- Which filament and why → `MATERIAL_DECISION_MATRIX.md`
- Slicer settings → `PRINT_SPEC.md`
- Sanding/paint/decals → `FINISHING_GUIDE.md`
- Putting it together → `ASSEMBLY_NOTES.md`
- Recording work → `PRINT_LOG_TEMPLATE.md` → `05_printed_parts_log/PRINT_LOG.md`

Raw model files live untouched in `unsorted_stl_raw/`; approved copies (with SHA-256
manifest) in `02_ready_to_slice/`. Claude Code session rules: `CLAUDE.md`.
Historical docs (build sheet v1/v2, print spec v2): `docs/`.
