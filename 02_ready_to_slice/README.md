# 02_ready_to_slice

COPIES of the selected, approved-for-printing model files, grouped into print-group
subfolders that mirror the build order in ../BUILD_SHEET.md.

Rules:
- Everything here is a copy. Originals live in ../unsorted_stl_raw/ and are never moved.
- Every copy MUST have a row in MANIFEST.md (original path + SHA-256) — that is the
  traceability chain. No manifest row → the file does not belong here.
- Do not put "maybe" parts here. Uncertain parts go to ../09_rejected_or_uncertain/REVIEW.md.
